기여하기 (Contributing)
======================

``espzero`` 프로젝트에 기여해 주셔서 감사합니다! 여러분의 도움으로 이 라이브러리가 더욱 발전할 수 있습니다.

기여 방법
--------

1. **버그 리포트**: 작동하지 않는 기능이나 에러가 있다면 GitHub Issue에 등록해 주세요.
2. **보드 프로필 추가**: 지원되지 않는 새로운 보드를 사용 중이라면, 프로필 파일에 보드 정보를 추가하여 Pull Request를 보내주세요.
3. **문서 개선**: 오타 수정이나 더 좋은 예제 코드가 있다면 언제든 환영합니다.

개발 환경 구축
-------------

1. 저장소를 포크(Fork)합니다.
2. 로컬에 클론(Clone)합니다.
3. 변경 사항을 작성한 후 테스트합니다.
4. 원본 저장소의 ``main`` 브랜치로 Pull Request를 생성합니다.
