Contributing
============

Thank you for your interest in contributing to the ``espzero`` project! Your help makes this library better for everyone.

How to Contribute
-----------------

1. **Bug Reports**: If you find something that doesn't work or an error, please register it as a GitHub Issue.
2. **Add Board Profiles**: If you are using a new board that isn't supported yet, add the board information to the profiles file and send a Pull Request.
3. **Improve Documentation**: Typo fixes or better example codes are always welcome.

Setting up the Development Environment
--------------------------------------

1. Fork the repository.
2. Clone it locally.
3. Make your changes and test them.
4. Create a Pull Request to the original repository's ``main`` branch.
