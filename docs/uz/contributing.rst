Loyiha rivojiga hissa qo'shish (Contributing)
==============================================

``espzero`` loyihasiga qiziqish bildirganingiz uchun tashakkur! Sizning yordamingiz ushbu kutubxonani hamma uchun yaxshiroq qiladi.

Qanday hissa qo'shish mumkin
---------------------------

1. **Xatolar haqida hisobot**: Agar ishlamaydigan biror narsa yoki xatoni topsangiz, iltimos, uni GitHub Issue sifatida ro'yxatdan o'tkazing.
2. **Plata profillarini qo'shish**: Agar hali qo'llab-quvvatlanmaydigan yangi platadan foydalanayotgan bo'lsangiz, profillar fayliga plata ma'lumotlarini qo'shing va Pull Request yuboring.
3. **Hujjatlarni yaxshilash**: Imlo xatolarini tuzatish yoki yaxshiroq misol kodlari har doim mamnuniyat bilan qabul qilinadi.

Ishchi muhitni sozlash
----------------------

1. Omborni fork qiling.
2. Uni mahalliy kompyuteringizga clone qiling.
3. O'zgarishlaringizni kiriting va ularni sinab ko'ring.
4. Asl omborning ``main`` tarmog'iga (branch) Pull Request yarating.
